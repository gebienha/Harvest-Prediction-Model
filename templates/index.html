Input Parameter Guide

Required Fields Explanation

Estimated Insects Count: Number of insects per square meter.
Crop Type: Crop category (0 or 1).
Soil Type: Soil category (0 or 1).
Pesticide Use Category: 1=Never, 2=Past, 3=Current.
Number of Doses per Week: Weekly pesticide applications.
Number of Weeks Used: Total weeks of pesticide use.
Number of Weeks Quit: Weeks since stopping pesticide use.
Season: Season index (1, 2, or 3).
